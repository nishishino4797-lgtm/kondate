# アプリの共有・公開ガイド

本アプリをカップル間や他のユーザーと共有するための手順をまとめます。

## 1. 🚀 Vercel へのデプロイ（Web公開）

作成したアプリをスマートフォンからアクセスできるように公開します。

1. **ソースコードの準備**: 現在のプロジェクトを GitHub にプッシュします。
2. **Vercel と連携**: [Vercel](https://vercel.com/) にログインし、GitHub リポジトリを接続します。
3. **環境変数の設定**: 
   - Vercel の project settings > Environment Variables を開きます。
   - `.env.local` にある `NEXT_PUBLIC_SUPABASE_URL` と `NEXT_PUBLIC_SUPABASE_ANON_KEY` をそれぞれ追加します。
4. **デプロイ**: 「Deploy」を実行すると、専用のURL（例: `https://our-menu.vercel.app`）が生成されます。

## 2. 👫 カップル間での共有

二人で全く同じデータをリアルタイム同期するには、**同じグループ名**でアクセスする必要があります。

- **共有URLの例**: `https://公開URL/?group=my-family`
- **仕組み**:
  - URLの最後に `?group=好きな名前` を付けてアクセスしてください。
  - 一度そのURLでアクセスすると、次回からはURLパラメータなしでもそのグループのデータが読み込まれます（ブラウザに保存されます）。
  - パートナーにも同じ `?group=...` 付きのURLを送ってあげてください。

## 3. 🛡️ 他のユーザーとのデータ分離

もし知人にアプリを勧めたい場合は、**別のグループ名**を伝えてください。
- あなたが `?group=my-family` を使い、友人が `?group=friend-family` を使えば、データが混ざることはありません。
- グループ名を指定せずにアクセスした場合は、全員共通の `couple1` グループに入ります。

## 📲 スマートフォンでの利用（PWA）
デプロイしたURLをスマホで開き、ブラウザの「ホーム画面に追加」を行うと、アイコンから一発で起動できるようになり、より快適に使用できます。
